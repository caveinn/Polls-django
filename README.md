# Polls-django
Polls site in django

**Technologies Used**
- Python
- Django
- Pipenv
- Postgres

**How to set up**
-
***Pre requisites***
1. Have postgres database installed
1. Have pipenv installed  

Clone the repo  
`git clone https://github.com/caveinn/Todo-django.git`  
Cd into the directory
`cd Todo-django`  
Run `pipenv shell`. This will set up a virtual environment, activate it and install the required dependencies
